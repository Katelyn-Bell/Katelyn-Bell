### 🚀 Featured Projects

#### 🛡️ [AgentSheriff](https://github.com/katelyn-bell/AgentSheriff) *(AI Agent Security Gateway)*
A security gateway engineered for autonomous AI agents to intercept and evaluate LLM tool calls. 
* **Stack:** Python (FastAPI), Next.js, Claude-Sonnet
* **Architecture:** Features a two-stage threat classifier utilizing deterministic regex heuristics and an LLM judge to dynamically generate and enforce strict execution policies based on agent `SKILL.md` configurations, mitigating prompt injections and destructive shell commands.

#### 🎓 [UKY Welcome Guide](https://github.com/katelyn-bell/UKY-Welcome-Guide) *(Campus Knowledge-Sharing App)*
A cross-platform mobile application facilitating targeted, secure information exchange among university students.
* **Stack:** React Native, TypeScript, Supabase, PostgreSQL
* **Architecture:** Engineered with Row-Level Security (RLS) for strict student data isolation. Implements `.edu` OTP account verification, real-time data synchronization for map-based location feeds, and moderated content pipelines.

#### 📄 [Syllabus Scanner](https://github.com/katelyn-bell/Syllabus-Scanner)
A full-stack application designed to automatically extract and organize course deadlines from academic documents.
* **Stack:** Python, React
